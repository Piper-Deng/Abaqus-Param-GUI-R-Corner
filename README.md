# Abaqus Parameter GUI (Minimal Public Release)

This repository contains a minimal public release with only:

- `Abaqus_Param_GUI_R_Corner.py`
- `README.md`

Usage:

1. Run `python Abaqus_Param_GUI_R_Corner.py`
2. Select your own Abaqus modeling script in the GUI
3. Run in CAE mode or noGUI INP export mode

Notes:

- Private modeling scripts are intentionally not included.
- Configure your Abaqus command/version in GUI before running.
